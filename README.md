# pm-toolkit
A personal project management toolkit built by Carolyn to simplify engineering and construction project management tasks.

## Current Features

- 📅 Calendar day calculator
- 📆 Business day calculator
- ➕ Add and subtract dates
- 📱 Mobile-friendly web version
- 💻 Desktop Python application

## Planned Features

### Schedule
- Contract duration calculator
- Notice to Proceed calculator
- Substantial Completion calculator
- Final Completion calculator
- Float calculator

### Cost
- Escalation calculator
- Contingency calculator
- Fee calculator
- Change Order calculator

### Engineering
- Unit conversions
- Water and wastewater calculations
- Tank volume
- Pipe velocity
- Pump horsepower

### Project Management
- Risk register helper
- Meeting minutes formatter
- Proposal utilities
- Daily report generator

## Technologies

- Python (Tkinter)
- HTML
- CSS
- JavaScript
- GitHub Pages

This project is being developed as a learning project while creating practical tools for engineering and construction management.
